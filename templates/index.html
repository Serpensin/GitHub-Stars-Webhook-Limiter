📝 Setup Instructions

  1. Add Repository: Fill in the form above with your repository URL, webhook secret, and Discord webhook URL.
  2. Generate Secret: Click "Generate" to create a secure random secret, then copy it.
  3. Configure GitHub Webhook:
    • Go to your GitHub repository → Settings → Webhooks → Add webhook
    • Set Payload URL to:
    • Set Content type to: application/json
    • Paste your secret in the "Secret" field
    • Select events: Stars and/or Watches
    • Ensure "Active" is checked
    • Click "Add webhook"
  4. Test: Star or watch your repository to verify Discord notifications work!

⚠️ Automatic Cleanup Policy

This service automatically removes:

  • Repositories: Deleted after 360 days of inactivity (no events received)
  • Invalid Webhooks: Repositories are removed if Discord webhook becomes invalid (checked every 28 days)
  • Deleted Repos: Repositories are removed if the GitHub repository no longer exists (checked every 28 days)
  • API Keys: Non-admin API keys are deleted after 360 days of inactivity

💡 Regular webhook events will keep your repository active. View system stats at /stats
